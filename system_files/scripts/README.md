# GRASP Photo Slideshow Builder

**Part of the GRASP (Genealogy Reports and Systems Programs) suite**

Scans a folder of family photo galleries and produces two files:

- `photo_config.json` ‚Äî structured index of every gallery and image, with optional caption notes
- `slideshow.html` ‚Äî a self-contained, no-server-needed viewer you can open in any browser

---

## Quick Start

```bash
# Scan your media folder and build the slideshow in one step
python build_slideshow.py all --media-dir ./media --title "Rose Family Photos"

# Open the result
open slideshow.html          # macOS
start slideshow.html         # Windows
xdg-open slideshow.html      # Linux
```

## Commands

### `scan` ‚Äî index photos ‚Üí `photo_config.json`

```bash
python build_slideshow.py scan \
  --media-dir ./media \
  --title     "Rose Family Photos" \
  --sort      name \
  --output    photo_config.json
```

| Flag | Default | Description |
|------|---------|-------------|
| `--media-dir` | `./media` | Root folder containing gallery sub-folders |
| `--title` | `Rose Family Photos` | Heading shown in the viewer sidebar |
| `--sort` | `name` | Sort order: `name`, `date`, or `random` |
| `--output` | `photo_config.json` | Where to write the config file |
| `--no-recursive` | *(off)* | Only scan top-level sub-folders |

### `build` ‚Äî `photo_config.json` ‚Üí `slideshow.html`

```bash
python build_slideshow.py build \
  --config photo_config.json \
  --output slideshow.html
```

### `all` ‚Äî scan + build in one step

```bash
python build_slideshow.py all \
  --media-dir  ./media \
  --title      "Rose Family Photos" \
  --config-out photo_config.json \
  --output     slideshow.html
```

## Adding Notes / Captions

After running `scan`, open `photo_config.json` in any text editor.
Each image entry looks like this:

```json
{ "src": "portraits/alice.jpg", "note": "" }
```

Fill in the `"note"` field for any image you want to caption:

```json
{ "src": "portraits/alice.jpg", "note": "Alice Rose, circa 1945. Taken at the family farm in Gainesville." }
```

Then run `build` again to regenerate the HTML with the new captions.
Notes appear as a gold-bordered banner at the bottom of the image
and can be toggled on/off while viewing.

---

## Tips

- The generated `slideshow.html` is **standalone** ‚Äî copy it anywhere alongside
  your `media/` folder and it works without a web server.
- Press **F11** in Firefox (or any browser) to hide the browser chrome and use
  the viewer in true fullscreen.
- Run `scan` again whenever you add new photos. The `"note"` fields you added
  to the config will be **overwritten** ‚Äî keep a backup of your annotated
  `photo_config.json` if you have many captions.
